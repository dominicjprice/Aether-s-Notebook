Decompress log is a java app that decompresses the files 
generated by the Andriod application "geotagger"/aether's notebook.

It can decompress DEFLATE and GZIP files using command line paraeameters.

To use

java -jar decompressLog FILENAME [gz|none]

To use with default compression standard:

java -jar decompressLog logfile.dfl 

To specify gzip compression:

java -jar decompressLog logfile.dfl gz


The output file is named FILENAME.log.
